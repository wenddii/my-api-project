# my-api-project
API for taskmanager

This is a simple api for a task manager app,on this day i am building the serializers and models.the model.py file contains two models, the user model imported from the built in auth user model and Task model containing different fields and using user as a foreign key.and the serializer i used modelserializer and on the view currently i just wrote the simplest view using view set and added permissions to some fields i plan to advance it with more work soon later.
